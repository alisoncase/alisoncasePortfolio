Geographic Name Proposal App

July 2024

This web map app prototype is designed to help streamline the process of submitting proposals for changes to domestic geographic names, based on the existing paper-based process managed by the U.S. Board on Geographic Names, a branch of the U.S. Geological Survey (USGS). USGS’s existing process published on its website requires the public to submit a PDF form via mail or email to USGS. This app leverages Esri.js, pulling data from Esri and USGS public REST services, for an interactive web map hosted on GitHub where users can look up data on geographic places while completing a digital version of the form developed in Esri Survey123. I developed the app as part of my Geospatial Web and Mobile Programming course at the University of Wisconsin–Madison.
